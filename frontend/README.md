
## Começando
Com o backend já inicializado e o Next js intalado e um novo terminal aberto, execute o servidor de desenvolvimento:

```bash
cd frontend

npm install

npm run dev

```

Abra http://localhost:3000 no seu navegador para ver o resultado.
